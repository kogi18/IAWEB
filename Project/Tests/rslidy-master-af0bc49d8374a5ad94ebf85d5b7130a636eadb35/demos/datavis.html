Open Data Formats

  • Spreadsheet
    • Excel (xls), OpenOffice (ods), comma-separated value (csv), JSON, etc.
    • machine-readable
    • (potentially) with a guaranteed URI
  • SPARQL + RDF
    • RDF triples in say XML notation.
    • machine-readable
    • (potentially) with a guaranteed URI
    • (potentially) live data = always current
    • RDF "Dialects" for Statistical Data:
      • The RDF Data Cube Vocabulary W3C initiative, based on SDMX ISO standard.
      • Simple Knowledge Organization System (SKOS) for concept hierarchies.
      • The Statistical Core Vocabulary (SCOVO) earlier project, now deprecated.

Tools for Visualising Data

Four varieties:

  • Web Services
    • must be online
    • point and click
    • can (sometimes must) share data
    • can (sometimes must) share finished visualisations
  • Standalone Applications
    • download, install, and use locally
    • point and click
    • generally produce static visualisations (limited interaction)
  • Scripting Languages (with Libraries)
    • download, install, and use locally
    • some basic programming
    • generally produce static visualisations (limited interaction)
  • Toolkits (with Programming APIs)
    • download source code or library and use locally (sometimes remote service via API)
    • much more programming
    • can produce fully interactive visualisations

A Eurostat Vis Tool

  1. Get RDF data cube from Eurostat wrapper.
  2. Query data using SPARQL, save subset as RDF.
  3. Load RDF subset into web browser.
  4. Generate JSON format on the fly in browser.
  5. Visualise using ProtoVis.
