README.md Markus Schofnegger, 2015 ## Introduction rslidy is a web-based presentation tool using HTML, CSS and JavaScript. It is able to display slides stored within a single HTML file. ## Installation To use rslidy for a presentation, the rslidy.js file and all relevant CSS files have to be included in your HTML file, which should also contain the presentation itself. Alternatively, the rslidy.min.js and rslidy-combined.min.css files can be used if no custom settings or other modifications are needed.
