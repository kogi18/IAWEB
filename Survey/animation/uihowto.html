Animation

Where to start planning?

  • In practice it is discussed at the end stages of the project
    • Seen as the end icing but should be part of the design
    • When everything is finished, it is too late to plan design with animation
    • A reason why animation is underestimated in design
  • Plan it as a part of design that has a set purpose
    • Start with planning already in early stages
    • Animation can replace a fixed design element
    • Add only animation that helps user and does not distract
    • In early stages story boards are useful
    • Understannd basic animation concepts (Disney 12) and how they effect the user
